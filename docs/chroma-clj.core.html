chroma-clj.core

A Clojure library that provides a simple interface to interact with ChromaDB, a document-oriented database.

The library includes functions to manage collections, perform CRUD operations on chroma documents, and utilize a special ChromaMap hashmap implementation with fuzzy key matching.
